# Access Control Matrix
Project: Zero Trust IAM & GRC Program
Author: Miguel Carrillo
Date: 2026-03-30

| Role       | Users            | Access Level       | MFA Required | Notes                      |
| ---------- | ---------------- | ------------------ | ------------ | -------------------------- |
| Admin      | IT Staff         | Full system access | Yes          | Can manage all accounts    |
| Manager    | Department Leads | Limited access     | Yes          | Only can approve requests  |
| Employee   | General staff    | Minimal access     | Optional     | Access only to needed apps |
| Contractor | External users   | Restricted         | Yes          | Temporary accounts         |
